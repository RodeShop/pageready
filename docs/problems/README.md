# Problems — Баг-тикеты

Mechanic берёт тикет → исправляет → закрывает (добавляет `## Решено`).

## Формат файла: `<YYYY-MM-DD>-<slug>.md`

```markdown
# [TICKET] Название проблемы

**Продукт:** <slug>
**Приоритет:** high / medium / low
**Статус:** open / in_progress / closed

## Проблема

[Что не так]

## Воспроизведение

[Шаги]

## Решено

[Что сделано, дата]
```
