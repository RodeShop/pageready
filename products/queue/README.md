# Queue — Очередь продуктов

Каждый файл = одна ниша в очереди на генерацию.

## Формат файла: `<slug>.md`

```markdown
# <Название продукта>

**Тип:** Notion Template | Prompt Bundle
**Ниша:** ...
**Аудитория:** ...
**Цена:** $...
**Приоритет:** A | B

## Почему эта ниша

[Обоснование: спрос, конкуренция, боль аудитории]

## Что должно быть внутри

- База 1: ...
- База 2: ...
- Ключевая фича: ...
```

## Статусы

- Файл в `queue/` — ждёт генерации
- MiMo взял → создал `products/draft/<slug>/`
- Готово → `products/ready/<slug>/`
