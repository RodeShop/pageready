# Agency Client OS

**Тип:** Notion Template
**Ниша:** Agency / Client Management
**Аудитория:** Small agencies, freelance studios (2-5 people)
**Цена:** $24
**Приоритет:** A
**Статус:** in_progress (MiMo начал, context limit — доделать)

## Почему эта ниша

Малые агентства ведут клиентов, проекты, задачи, предложения, инвойсы и команду в 4-5 разных инструментах. Notion заменяет всё в одном месте. Конкурентов мало ($24 оправдан).

## Что должно быть внутри (6 баз)

- **Clients** — статус, контакты, выручка, связь с Projects
- **Projects** — дедлайн, бюджет, статус, связь с Tasks + Clients
- **Tasks** — приоритет, исполнитель, дедлайн, Overdue формула
- **Proposals** — статус (Draft/Sent/Won/Lost), сумма, связь с Client
- **Invoices** — статус оплаты, сумма, due date, связь с Project
- **Team** — роль, скилы, загрузка, связь с Tasks

## Rollup chains

Tasks → Projects (Task Count, Done Count)  
Projects → Clients (Total Budget, Active Projects)  
Invoices → Clients (Total Invoiced, Unpaid Amount)
